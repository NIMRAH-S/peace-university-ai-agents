# Peace University AI Agents

## Project Overview

This project aims to build an AI-powered multi-agent system for Peace University to automate the marketing and student lead management process.

## Phase 1

Lead Capture Agent

### Objective

Capture leads from different sources, validate the data, remove duplicates, and store them in a centralized Google Sheet for further processing.

## Technology Stack

- n8n
- Docker
- Google Sheets API
- Google Forms
- Google Drive
- GitHub
